# Pet shop Amigo Fiel
Um projeto web moderno e responsivel para pet shop amigo fiel, desenvolvido com foco em boas praticas de desenvolvimento Front-End, escabilidade e experiencia, ideial para quem buscar um ponto de partida solido para projetos web institucionais.

---

## ✨ Funcionalidades
- **HomePage** com hero section e chamada para acao
- **Secao de Servicos** (banho, tosa, veterinario, etc..)
- **Galeria de Fotos**
- **Formulario de contato** com validacao em javascript
- **secao de localizacao**
